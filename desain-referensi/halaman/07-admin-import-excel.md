## Komponen
- **Info box:** daftar kolom yang diharapkan, warna info
- **Bulan/Tahun selector:** wajib dipilih sebelum upload
- **Drag & drop zone:** dashed border, highlight saat hover file
- **Progress bar:** Bootstrap progress bar, animated striped
- **Hasil Summary:** card success/warning
- **Error Table:** daftar baris error dengan keterangan jelas
- **Validasi:**
  - Cek format file (.xlsx, .xls)
  - Cek header kolom persis sesuai spesifikasi
  - Cek CUST ada di tabel anggota
  - Cek GRUP ada di tabel jenis_potongan
  - Cek field numerik valid (PINJ, AWAL, ANGS, dll)

## Responsif
- Drag zone: full-width di mobile
- Error table: horizontal scroll di mobile
