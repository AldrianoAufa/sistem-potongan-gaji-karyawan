# 📐 Referensi Desain — Sistem Potongan Gaji PT Primatex Indonesia

Folder ini berisi panduan desain untuk setiap halaman website.
Gunakan file-file di folder ini sebagai referensi saat membuat desain di **Google Stitch**.

## Struktur Folder

```
desain-referensi/
├── README.md               ← File ini
├── panduan-umum.md          ← Warna, tipografi, komponen umum
├── halaman/
│   ├── 01-login.md
│   ├── 02-admin-dashboard.md
│   ├── 03-admin-karyawan.md
│   ├── 04-admin-jabatan.md
│   ├── 05-admin-jenis-potongan.md
│   ├── 06-admin-input-bulanan.md
│   ├── 07-admin-import-excel.md
│   ├── 08-admin-laporan.md
│   ├── 09-user-dashboard.md
│   └── 10-user-potongan.md
└── hasil-stitch/            ← Simpan hasil desain dari Google Stitch di sini
    └── .gitkeep
```

## Cara Penggunaan

1. Baca `panduan-umum.md` untuk warna, font, dan komponen dasar
2. Buka file halaman yang ingin didesain di folder `halaman/`
3. Buat desain di Google Stitch berdasarkan spesifikasi yang ada
4. Simpan hasil screenshot/export di folder `hasil-stitch/`
5. Hasil desain akan digunakan sebagai acuan saat implementasi
